What is SESL?

SESL is a forward-chaining expert-system language designed for deterministic, repeatable reasoning. It transforms documents or structured descriptions into rules, facts, and transparent logic flows.

Every SESL answer comes with a detailed explanation showing which rules fired, why they fired, and how the final result was produced. SESL can run inside ChatGPT (via MCP) or standalone on a local machine.

  • Readable by non-programmers
  • Deterministic & reproducible
  • Traceable & audit-friendly
  • Safe under malformed input

Why SESL exists

LLMs produce fluent answers but are inherently probabilistic. The same question may yield different responses, and hallucinations remain an unavoidable risk.

SESL eliminates this uncertainty by relying on explicit rules and facts. The output is always:

  • Predictable
  • Consistent
  • Auditable
  • Scientifically traceable

SESL Command Line Interface

The SESL CLI allows full rule editing, linting, debugging, and scenario execution outside ChatGPT.

  • Load & save SESL files
  • Inspect rules, facts, and scenarios
  • Run scenarios (normal or ‚Äúdescribed‚Äù)
  • Inline and multiline rule editing
  • Full linting with error/warning reporting
  • Dependency graph generation
# Run a SESL file
sesl rules.sesl

# Run with explanation
sesl describe rules.sesl

# Lint your rules
sesl lint rules.sesl

# Run one scenario
sesl run rules.sesl --scenario Case1
